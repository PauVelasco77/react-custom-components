# Button

The `Button` component is used to trigger actions and can be customized with various properties.

## Usage

To use the `Button` component in your project, import it from the components library and use it as shown below:

```tsx
import { Button } from "components-library";

const Example = () => {
  return (
    <Button
      text="Press me"
    />
  );
};

export default Example;
